# HireLens – Interview Analysis Tool

HireLens is a project designed to help analyze interview responses using technology. The goal is to assist in evaluating candidate communication, clarity, and response structure during interviews.

The system records interview answers using microphone input and processes the responses using Python-based analysis.

Features
- Voice input for recording interview responses
- Basic processing of responses
- Simple interface for recording and reviewing answers
- Designed as an experimental tool for interview evaluation

Technologies Used
- Python
- Speech recognition
- Basic UI for recording responses
